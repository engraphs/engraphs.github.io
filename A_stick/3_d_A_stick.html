6_A_stick
You can download as an example the application taskW which runs in a taskwindow an executable that is dragged to it .

7_A_stick
Its !Drag file is a TaskObey file ( filetype &FD7 ) which runs its first commandline argument ( %0 ) which is always the pathname of the object dragged onto the iconbar icon .

8_A_stick
Its !Run file 's essential command is stick <Obey$dir> which creates the application and puts its icon on the iconbar .
