9_A_stick
Every sticky application of this kind must have a template file called Templates , with a window - template called info in it .

10_A_stick
This is to give the application an iconbar menu that behaves properly .

11_A_stick
The simplest way of creating the template file is to copy the one in !taskW and edit it in a template - editor .
